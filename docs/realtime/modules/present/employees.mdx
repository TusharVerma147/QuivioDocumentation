
# Employees

The Employees Screen provides detailed performance metrics for individual employees, helping users track and evaluate their contributions.

<figure> <img src="/img/inventory/employees.png" alt="This is an example image  of Employees" width="300" height="600" /></figure>

## Overview

The Employees Screen allows users to:

- View detailed metrics for each employee.
- Highlight achievements and identify areas for improvement.

## Key Features

- **Employee Profiles**: Displays detailed stats for individual employees.
- **Metrics**: Tracks performance indicators like hours worked and sales contributions.
- **Filters**: Allows filtering by team or role.

## Code Structure

### Imports

The screen imports several utilities and components such as:

- **Profile Card Components**:For displaying employee details.
- **Redux**: employeeMetricsReducer for managing state.


- **refreshEmployeeMetrics**: Fetches updated metrics from the server.
