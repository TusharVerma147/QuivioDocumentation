
# About Us

The About Us functionality in the Settings screen allows users to access information about the application and its developers. This document outlines the key features, structure, and functionality of the About Us process.

## Overview

The About Us functionality provides the following features:
- It is designed to provide users with a clear and concise overview of the application. 
- It ensures that users can easily access this information through a dedicated screen, which is seamlessly integrated into the application's navigation flow.
-  The use of a WebView component allows for dynamic content loading, ensuring that the information remains up-to-date.



## Key Features

- **Information Display**: The component displays a brief description of the application and its developers.
- **Navigation**: When the user selects the About Us option, the application navigates to a screen that provides more detailed information.
- **WebView Integration**: The detailed information is displayed using a WebView component, which loads content from a specified URL.


## Code Structure

### Imports

The functionality imports several utilities and components, such as:

- **Custom Components**: CustomLoader.
- **Hooks**: useAppSelector, useAppThunkDispatch.
- **Redux**: clearToken.
- **Services**: AnalyticsService, ModalController.
- **Utilities**: screenNames, network.


### State Management

The component uses the useState hook for state management:

```javascript
const [webViewLoading, setWebViewLoading] = useState(true);
```

### Event Handlers

- **onPressOption**: Triggers the navigation to the About Us screen when the About Us option is selected.


### Navigation

Navigates to the moreInfo screen with the necessary parameters to display the About Us information.
