
# Best Sellers

The Best Sellers component provides a list of the top-selling items, allowing users to analyze sales performance over a specific period.

<figure> <img src="/img/inventory/historicalSellers.png" alt="This is an example image  of best Sellers" width="300" height="600" /></figure>

## Overview

The Best Sellers component allows users to:

- View the top-selling items.
- Analyze sales data and revenue generation.
- Apply date range filters to view targeted sales data.

## Key Features

- **Top-Selling Items**:  Displays items ranked by total sales and revenue.
- **Key Metrics**: Includes item details like sales count and revenue.
- **Custom Date Filters**: Allows filtering by date range for targeted analysis.

## Code Structure

### State Management

```javascript
const initialState = {
  topGreetersData: [],
  startDate: '',
  endDate: '',
  isLoading: false,
};
const [state, updateState] = useSimpleReducer(initialState);
```

### Event Handlers

- **handleDateChange**: Filters greeter data based on the selected date range.
- **handleRefresh**: Refreshes the greeter rankings dynamically.

### API Integration

- **getTopGreetersAPI**: Fetches data for the top greeters over a specified period.
