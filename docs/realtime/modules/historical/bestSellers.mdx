
# Best Sellers

The Best Sellers component in the Historical Screen provides a detailed breakdown of the top-performing products or services at a selected location, offering insights into their performance metrics such as quantity sold and net sales.

<figure> <img src="/img/inventory/historicalSellers.png" alt="This is an example image  of best Sellers" width="300" height="600" /></figure>

## Overview

The Best Sellers component allows users to:

- View the top-performing products or services based on quantity sold and net sales.
- Analyze trends and performance over time.
- Apply date range filters for custom analysis.

## Key Features

- **Top Performers**: Displays the best-selling products or services based on quantity sold and net sales.
- **Custom Date Filters**: Enables custom date selection for analysis.
- **Dynamic Data**: Updates data based on the selected date range and location.

## Code Structure

### State Management

```javascript
const initialState = {
  bestSellerData: [],
  selectedStartDate: '',
  selectedEndDate: '',
  isBestSellerLoading: false,
};
const [state, updateState] = useSimpleReducer(initialState);
```

### Event Handlers

- **handleDateChange**: Filters best sellers data based on the selected date range.
- **handleRefresh**: Refreshes the data dynamically.

### API Integration

- **getBestSellersData**: Fetches best sellers data for the selected date range.

## Analytics

## Navigation

The Best Sellers component is accessible via the tab-based navigation in the Historical Screen.
